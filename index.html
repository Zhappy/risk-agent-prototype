# 舆情数据查询 Skill

## 功能描述
从舆情数据库中查询指定时间范围内的舆情数据。

## 输入参数
- start_date: 开始日期 (YYYY-MM-DD)
- end_date: 结束日期 (YYYY-MM-DD)
- keywords: 关键词列表 (可选)
- channels: 渠道筛选 (可选)
- sentiment: 情感倾向筛选 (可选)

## 输出格式
```json
{
  "total": 120,
  "records": [...]
}
```
